Why create this App ?

On the discourse site,they said: "No app required,Discourse is designed from the beginning for high resolution touch devices." So why need this App?
I love Discourse, But discourse heavily dependent on JavaScript; And the JS performance on android devices is so much worse, Sometimes it crash the Browsers(More info: Why is Discourse so slow on Android? ). So right now I decide create a Native app for my android phone.

My password safe ?

Yes! Using the AES encryption algorithm to protect your password!

Sometimes the app crashed

Now this App have some bugs, if crashed or something goes wrong, please reopen it. you can use the Feedback menu to help us fix it.

Older Android devices support

Currently old(pre-4.0,Ice Cream Sandwich)OS versions are not supported. We will investigate supporting older versions in the future.

Discourse compatible versions?

The application compatible with Discourse 0.9.X + version. The 0.8.X- versions not supported. Please update your discourse site.
